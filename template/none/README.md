# Vite WebGPU Project

A minimal Vite + TypeScript project with an initialized WebGPU canvas and no rendered content.

## Run it

```sh
npm install
npm run dev
```

Open the local URL printed by Vite in a current browser with WebGPU support, then add rendering code in `src/renderer/Renderer.ts`.

## Commands

- `npm run dev` starts the development server.
- `npm run build` type-checks and creates a production build in `dist/`.
- `npm run preview` serves the production build locally.
